# Quick Start: Theme Colors

## 🎨 Change Your School Colors in 3 Steps!

---

### Step 1: Open School Settings
```
1. Login as Admin/Super Admin
2. Click "Settings" in top menu
3. Click "School Settings"
4. Scroll to "Theme Colors & Branding" section
```

---

### Step 2: Choose Your Theme

#### Option A: Use a Preset (Easy)
```
1. Click the "Choose Theme Preset" dropdown
2. Select one of 8 themes:
   • Default (Bootstrap Blue)
   • School Green
   • Royal Blue
   • Purple Academic
   • Orange Energy
   • Teal Modern
   • Crimson Tradition
   • Navy Professional
3. Watch the preview buttons change color!
```

#### Option B: Custom Colors (Advanced)
```
1. Select "Custom Colors" from dropdown
2. Click each color picker to choose:
   • Primary Color (main brand)
   • Secondary Color (supporting)
   • Success Color (confirmations)
   • Info Color (highlights)
   • Warning Color (alerts)
   • Danger Color (errors)
3. See live preview update!
```

---

### Step 3: Save & Refresh
```
1. Click "Save Settings" button
2. Press Ctrl+F5 (or Cmd+R on Mac) to refresh
3. Enjoy your new theme!
```

## 💡 Quick Tips

### Matching Your Logo:
```
1. Look at your school logo
2. Pick the dominant color
3. Set it as Primary color
4. Keep other colors standard
```

### Testing Colors:
```
After saving, check:
- Dashboard looks good ✓
- Buttons are readable ✓
- Alerts are visible ✓
- Forms look clear ✓
```

### Reset to Default:
```
1. Choose "Default (Bootstrap Blue)"
2. Save Settings
3. Done!
```

---

## 🆘 Troubleshooting

**Colors not showing?**
→ Press Ctrl+F5 to hard refresh

**Want to change again?**
→ Just go back and select different colors

**Made a mistake?**
→ Select a preset to reset, then customize
