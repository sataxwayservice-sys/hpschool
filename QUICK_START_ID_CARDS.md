# Quick Start Guide - Student ID Cards

## 🚀 Get Started in 3 Steps

### Step 1: Access the Generator
Choose one of these methods:

**Method A - From Navigation**
1. Click **Students** menu (top nav bar)
2. Click **Generate ID Cards**

**Method B - From Student List**
1. Go to **Students → View Students**
2. Click **Generate ID Cards** button (top right)

**Method C - Single Student**
1. Open any student profile
2. Find **Quick Actions** section
3. Click **Generate ID Card**

---

### Step 2: Select Options

**Choose Design Template:**
- **Default** - Standard horizontal card (best for general use)
- **Modern** - Colorful gradients (eye-catching)
- **Professional** - Blue corporate style (formal)
- **Colorful** - Rainbow borders (fun, vibrant)
- **Vertical** - Tall badge style (for lanyards)

**Filter Students (Optional):**
- Select **Class** (e.g., Class 10)
- Select **Section** (e.g., Section A)
- Or leave blank for all students

Click **Filter** to preview

---

### Step 3: Print!

1. Click **Print Cards** button
2. Browser print dialog opens
3. **Print Settings**:
   - Paper: A4
   - Quality: Best/High
   - Orientation: Portrait
4. Click **Print**

---

## 📋 Printing Tips

### For Best Results

**Paper:**
- Use **cardstock** (200-300 GSM)
- Glossy finish for professional look
- Matte finish for everyday use

**Printer Settings:**
- Quality: **Best** or **High**
- Color: **Full color**
- Paper type: **Cardstock/Heavy**

**Finishing:**
- Cut with paper cutter for clean edges
- **Laminate** for durability (recommended)
- Punch hole for lanyard (vertical design)

## 💰 Cost-Effective Tips

### Budget-Friendly Options

**Option 1: In-House Printing**
- Print on regular paper first (preview)

- DIY lamination pouches
- **Cost**: ~₹5-10 per card

**Option 2: Local Print Shop**
- Generate at home

- Professional lamination
- **Cost**: ~₹15-25 per card

**Option 3: Bulk Printing**
- Print entire class at once
- Use professional printer
- Bulk lamination discount
- **Cost**: ~₹10-15 per card (bulk)

## 🔧 Common Issues & Solutions

### Problem: Photo Not Showing
**Solution:**
- Upload student photo first (Edit Student page)
- Supported formats: JPG, PNG
- Default avatar shows if no photo

### Problem: Cards Too Small/Large
**Solution:**
- Use browser zoom (Ctrl + / Ctrl -)
- Check printer scaling settings
- Set to "Actual size" not "Fit to page"

### Problem: Colors Look Different
**Solution:**
- Use Chrome browser (best results)
- Check printer color settings
- Preview before final print

### Problem: Cards Not Aligned
**Solution:**
- Check printer margins (should be minimal)
- Use high-quality paper
- Test print on regular paper first

## 📱 QR Code/Barcode

Each ID card includes a barcode with the student's admission number:
- **Format**: `*ADMISSION_NO*`
- **Use**: Quick verification with barcode scanner
- **Benefits**: Fast attendance, library checkout, etc.

## 🎯 Best Practices

### For Schools

1. **Standardize**: Choose one design template for all students
2. **Quality Photos**: Take professional photos during admission
3. **Batch Process**: Print class-wise for efficiency
4. **Laminate All**: Protect cards from wear and tear
5. **Track Validity**: Include academic year on card
6. **Replacements**: Have a process for lost cards

### For Students

1. **Daily Wear**: Encourage students to wear cards daily
2. **Care**: Teach proper card handling
3. **Lanyard**: Provide lanyards (vertical design)
4. **Report Loss**: Immediate reporting of lost cards
5. **Keep Clean**: Wipe cards regularly

## 📏 Card Dimensions

### Horizontal Cards
- **Width**: 85.6mm (3.37 inches)
- **Height**: 54mm (2.13 inches)
- **Same as**: Credit card size
- **Cards per A4**: 8-10

### Vertical Cards
- **Width**: 60mm (2.36 inches)
- **Height**: 90mm (3.54 inches)
- **Same as**: Badge size
- **Cards per A4**: 6-8

## 🌟 Examples

### Example 1: Generate Cards for Class 10-A
```
1. Students → Generate ID Cards
2. Design: Professional
3. Class: 10
4. Section: A
5. Filter → Preview shows 45 students
6. Print Cards → 5 pages (A4)
7. Print on cardstock
8. Laminate all cards
```

### Example 2: Replace Lost Card
```
1. Find student profile
2. Generate ID Card (Quick Actions)
3. Design: Same as original (Professional)
4. Print Cards
5. Print single card
6. Laminate
7. Issue to student
```

### Example 3: New Batch of Students
```
1. Upload all photos first
2. Generate ID Cards
3. Design: Default
4. No class filter (all new students)
5. Print Cards
6. Bulk print at shop
7. Laminate all
8. Distribute during orientation
```
