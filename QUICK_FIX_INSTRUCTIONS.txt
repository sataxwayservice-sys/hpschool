================================================================================
  TROUBLESHOOTING
================================================================================

If you still see errors after uploading:

Problem: "Database connection failed"
Solution:
  1. Double-check credentials in config/database.php
  2. Verify database exists in InfinityFree phpMyAdmin
  3. Check database was imported successfully
  4. Visit: yoursite.infinityfree.com/test_login.php

Problem: "CURRENCY_SYMBOL warning"
Solution:
  1. Clear browser cache (Ctrl + Shift + R)
  2. Verify config/config.php was uploaded
  3. Check file permissions
